# Lorus Bot

Телеграм-бот для заработка звёзд через выполнение заданий.

## Возможности
- Выполнение локальных и субграм-заданий
- Реферальная система
- Мини-игра для получения звёзд
- Админ-панель для управления заданиями и балансами

## Технологии
- Python 3.11 + aiogram 3.x
- PostgreSQL
- Linux deployment (systemd, Nginx)

## Запуск
```bash
pip install -r requirements.txt
python main.py
